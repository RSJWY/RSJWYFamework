# Unity License 系统使用说明

本系统包含两部分：
1. **Python 脚本** (`gen_license.py`): 用于生成 RSA 密钥对和签发 License 文件。
2. **Unity 脚本** (`LicenseVerifier.cs`): 用于在游戏中验证 License 的合法性。

---

## 第一步：环境准备 (Python)

确保你安装了 Python，并安装了必要的加密库 `pycryptodome`。

在终端/命令行中运行：
```bash
pip install pycryptodome
```

---

## 第二步：生成 RSA 密钥对

我们需要一对公钥和私钥。私钥用于生成 License（由你保管），公钥用于验证 License（放入 Unity 项目）。

1. 打开终端，进入 `license脚本` 目录。
2. 运行以下命令生成密钥：
   ```bash
   python gen_license.py --gen-keys
   ```
3. 执行成功后，目录下会生成两个文件：
   - `private_key.pem` (私钥：**严禁泄露**，请妥善保管)
   - `public_key.pem` (公钥：用于 Unity 客户端)

---

## 第三步：Unity 端配置

1. **挂载脚本**: 
   - 在 Unity 场景中创建一个空的 GameObject（例如命名为 `LicenseManager`）。
   - 将 `Assets/Scripts/License/LicenseVerifier.cs` 脚本挂载上去。

2. **填入公钥**:
   - 用文本编辑器打开刚才生成的 `public_key.pem`。
   - 复制 **全部内容** (包括 `-----BEGIN PUBLIC KEY-----` 和 `-----END PUBLIC KEY-----`)。
   - 在 Unity Inspector 中，找到 `LicenseVerifier` 组件，将内容粘贴到 **Public Key PEM** 字段中。

---

## 第四步：获取机器码

为了签发绑定机器的 License，你需要先获取目标设备的机器码。

1. 运行 Unity 场景（或者不运行，直接在编辑器模式下）。
2. 在 Inspector 中，右键点击 `LicenseVerifier` 组件的标题栏。
3. 选择 **"复制机器码到剪贴板" (Copy Machine Code to Clipboard)**。
4. 现在你的剪贴板里已经有了机器码（通常是一串哈希值）。

---

## 第五步：生成 License 文件

回到 Python 脚本所在的终端。

**交互模式 (推荐)**:
1. 直接运行 `python gen_license.py`。
2. 选择菜单中的 `2. 生成 License`。
3. 粘贴刚才复制的 **机器码**。
4. 输入有效期（默认 30 天）。
5. 文件名直接回车使用默认值 `License.lic`。

**命令行模式**:
```bash
# 示例：给机器码 "你的机器码" 签发 365 天的 License
python gen_license.py --code "你的机器码" --days 365
```

---

## 第六步：部署 License

1. 将生成的 `License.lic` 文件复制到 Unity 项目的 `Assets/StreamingAssets/` 文件夹中。
   - 如果 `StreamingAssets` 文件夹不存在，请在 `Assets` 下手动创建一个。
2. **注意**: 在打包发布后，License 文件通常放置在 `StreamingAssets` 目录中，或者由玩家手动放置在 `PersistentDataPath` 下。

---

## 第七步：验证

1. 运行 Unity 场景。
2. 观察 Console 控制台日志。
   - 成功: `[LicenseVerifier] License 有效。`
   - 失败: 会显示具体的错误信息（如“机器码不匹配”、“License 已过期”等）。
3. 你可以在代码中订阅 `OnLicenseValid` 事件来进入游戏，或订阅 `OnLicenseInvalid` 事件来弹出错误提示。

```csharp
// 代码示例
public LicenseVerifier verifier;

void Start() {
    verifier.OnLicenseValid += StartGame;
    verifier.OnLicenseInvalid += ShowError;
}
```
